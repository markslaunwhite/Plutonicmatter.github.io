Divotee


Divotee is a booking patform for golf players in South East Asia. It allows players to choose a caddy based on experience, language and overall rating by other users. The app is fully responsive from web to mobile and features a clean and easy to use UI.


Role:

Design, UX, UI, Development


Tech Used:

HTML/CSS, jQuery, JS, Ruby on Rails, Bootstrap
